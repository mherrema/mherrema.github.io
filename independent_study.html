Independent Study

I will be continuing to work on PlanItUp for an independent study while attending GVSU. The current status supports the following:

  • "MEAN" Stack structure

  • Authenticated login

  • Limited timetracking

  • Creation of projects, tasks, and discussions

  • User interface for navigating the projects, tasks, and discussions

  • User profile pictures stored on Amazon S3 server

  • A feed of updates relating to the user's projects displayed on dashboard

Further Improvement

While the application is fairly built out, there is a good amount to still be implemented:

  • Consistent creation and deletion of all items

  • Full timetracker implementation

  • Move user profile images to an Amazon CloudFront server

  • Messaging between users

  • Email notifications - server running in background, not sleeping

  • Live chat - using sockets

  • CSS Cleanup

  • Allow uploading of files, tagging to projects, tasks, or discussions

  • Develop API to support a "bearer" token instead of having a session ID to create a more "RESTful" API
